# RefreshControl 下拉刷新

滚动容器的下拉刷新功能，配合 ListView 使用时，需将 RefreshControl 放到 header 位置

## 安装

```bash
$ npm install rax-refreshcontrol --save
```

## 引用

```jsx
import RefreshControl from 'rax-refreshcontrol';
```

## 属性

| 名称         | 类型     | 默认值  | 描述        |
| :--------- | :----- | :--- | :-------- |
| refreshing | String |      | 是否显示      |
| onRefresh  | Number |      | 监听下拉刷新的行为 |
